# Senior Quote Puzzle 🧩

A math puzzle that leads to my senior quote. The number on the page is encrypted — can you figure out the answer?

**Hint:** The formula is the Fibonacci recurrence, and the number has a secret base.

👉 [https://sean-shmulevich.github.io/SeniorQuote/](https://sean-shmulevich.github.io/SeniorQuote/)

## The Puzzle

The number `1,288,191,304` converted from base 10 to base 36 gives the answer. The key `2-0-1-9` (graduation year) was used as a Caesar cipher hint in the original version.
